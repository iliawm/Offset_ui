# 🧩 Offset UI

A modern and minimal UI component library built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. This project aims to provide ready-to-use, customizable components featuring an "Offset" design style (characterized by layered shadows and distinct depth).

## ✨ Features

- **⚡ Next.js App Router**: Leveraging the latest Next.js structure for optimal performance and SSR.
- **🛡️ Type-Safe**: Fully typed codebase to minimize runtime errors.
- **🎨 Tailwind CSS**: Utility-first styling for rapid customization.
- **🧩 Modular Architecture**: Components are isolated and reusable.
- **📦 PNPM**: Fast, disk-space efficient package management.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (83.7% of codebase)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Package Manager:** [PNPM](https://pnpm.io/)
